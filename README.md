# ⏱ TimeSync

כלי אוטומטי להזנת שעות עבודה לאתר Synerion מתוך קובץ אקסל.

> ⚠️ **חשוב לדעת**
>
> - הכלי מזין שעות כניסה ויציאה בלבד.
> - הכלי אינו שולח או מאשר דוחות נוכחות.
> - חופשות, מחלות, חגים וכל סוג היעדרות אחר יש להזין ידנית באתר.

---

## 📥 הורדת הפרויקט

1. לחץ על הכפתור הירוק **Code**.
2. בחר **Download ZIP**.
3. חלץ את קובץ ה־ZIP לתיקייה במחשב.

---

## 🚀 הפעלה ראשונה

1. לחץ פעמיים על `run.bat`.
2. הזן את הפרטים הבאים:
   - שם משתמש (תעודת זהות)
   - סיסמה
   - קישור לעמוד הזנת הנוכחות

   לדוגמה:

   ```
   https://company.synerioncloud.com/SynerionWeb/#/dailyBrowser
   ```

3. בחר את קובץ האקסל שברצונך לטעון.
4. הסקריפט יפתח את הדפדפן ויזין את הנתונים באופן אוטומטי.
5. בסיום התהליך יש לעבור על הנתונים ולאשר אותם באתר.
6. במידה וקיימים ימי חופשה, מחלה, חג או היעדרויות אחרות – יש להזין אותם ידנית.

> בפעמים הבאות אין צורך להזין את פרטי ההתחברות מחדש — פשוט הפעל את `run.bat`.

---

## 📄 הורדת קובץ מאתר המרכבה

1. היכנס לדוח הנוכחות באתר המרכבה.
2. לחץ על שלוש הנקודות בתחתית המסך (ליד כפתור **שליחה לאישור**).
3. בחר **הורד לקובץ**.
4. שמור את קובץ האקסל במחשב.
5. בחר את הקובץ בעת הפעלת TimeSync.

---

## 📋 דרישות מערכת

- Windows 10 / 11
- Python מותקן במחשב

https://www.python.org/downloads/

## ⚠️ מגבלות

אין לכלול בקובץ האקסל ימים הכוללים:

- חופשה
- מחלה
- חג
- היעדרות מיוחדת

ימים אלו יש להזין ידנית באתר Synerion.

---

## ⚙️ הגדרות (`.env`)

בפעם הראשונה שהכלי מופעל נוצר קובץ `.env` באופן אוטומטי.

ניתן לערוך אותו ידנית במידת הצורך:

```env
SYNERION_USERNAME=123456789
SYNERION_PASSWORD=your_password
ATTENDANCE_URL=https://company.synerioncloud.com/SynerionWeb/#/dailyBrowser
```
