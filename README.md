# ⏱ TimeSync

כלי אוטומציה להזנת שעות עבודה לאתר Synerion מתוך קובץ אקסל.

> **⚠️ חשוב לדעת:**
> - הכלי **מזין שעות כניסה ויציאה בלבד** — הוא לא שולח או מאשר דוחות
> - הכלי **לא מטפל** בחופשות, ימי מחלה, חג, או כל סוג היעדרות אחר — יש להזין אותם ידנית באתר

---

## שכפל את הפרויקט

1. לחץ על הכפתור הירוק '<> Code' ובחר ב-Download ZIP

## 🚀 הפעלה מהירה

1. חלץ את ה-ZIP לתיקייה כלשהי
2. לחץ פעמיים על `run.bat`
3. **בפעם הראשונה בלבד** — הזן שם משתמש, סיסמה, וקישור לאתר
4. בחר את קובץ האקסל שלך בחלון שייפתח
5. הסקריפט יפתח דפדפן ויזין את השעות אוטומטית

> **הפעמים הבאות** — לחץ `run.bat` בלבד, הפרטים כבר שמורים.

## 📋 דרישות מקדימות

- **Windows בלבד**
- **Python מותקן** — [הורד מכאן](https://www.python.org/downloads/)
  - בזמן ההתקנה: סמן ✅ **"Add Python to PATH"**
- חיבור לאינטרנט בהתקנה הראשונה (להורדת חבילות)

---

> ⚠️ ימים עם חופשה / מחלה / חג — **אל תכלול אותם באקסל**, הזן אותם ידנית באתר לפני או אחרי הרצת הסקריפט.

---

## ⚙️ הגדרות (קובץ `.env`)

הקובץ נוצר אוטומטית בפעם הראשונה. ניתן לערוך ידנית בכל עורך טקסט:

```
SYNERION_USERNAME=your_username
SYNERION_PASSWORD=your_password
ATTENDANCE_URL=https://...
```

`HEADLESS=true` — הדפדפן יעבוד ברקע ללא חלון גרפי

> ⚠️ קובץ `.env` מכיל סיסמאות — **אל תשלח אותו לאף אחד ואל תעלה אותו לאף אחד!**
